# Links to Leads

Landing page for golf fitness trainers — fill your fairway with premium clients.

## Preview locally

```bash
cd ~/Projects/links-to-leads
python3 -m http.server 8765
```

Open [http://localhost:8765](http://localhost:8765)

## Files

- `index.html` — page structure & copy
- `styles.css` — design system (golf green + gold)
- `script.js` — form placeholder + scroll animations
